Creating Custom Controls for Quick Forms

You can create custom controls and use them in Application Architect Quick Forms.

Summary

  1. Start with an ASP.Net control. If you start with a browser-side, JavaScript-based control, then write the server-side piece to make it an ASP.Net control.
  2. Write an Application Architect Quick Form design-time control for the Quick Form Designer. This can be non-trivial for some controls, especially if you need custom property sheet editors in the Quick Form Designer.
  3. Write an NVelocity template to support the Quick Form generation process.
  4. Modify a couple files in the web project so that the control is usable and so Quick Form code generation will work.
  5. To support the Web Form Designer, create an editor for the control:
    1. Write a Web Form control. This involves a few JavaScript files.
    2. Modify a couple files to add your new control to the list of available Web Form Designer controls.

For those who have access to GitHub, consider searching for sample projects which might give additional detail.
