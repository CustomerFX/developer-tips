Using Policy Activities

Policy activities in Windows Workflow allow a workflow designer to set up complex business rules based on conditions and actions.

To create a policy activity

  1. From the Visual Studio toolbox under Windows Workflow v3.0, drag the Policy item into the designer of your new workflow.

  2. In the Property grid, click on the RuleSetReference property box, and then click the ellipsis button.
    The Select Rule Set dialog box opens.

  3. Add your business rules. Example.

    Example

    1. In the Select Rule Set dialog box, click New, and then click Add Rule.

    2. Give the rule a name, for example, AZLeads.

    3. Type a condition in the If box, for example:
      CurrentEntity.Address.State = "AZ"

    4. In the Then Actions box, type an action, for example:
      CurrentEntity.Owner = Teams.Southwest

    5. In the Else Actions box, type an alternate action, for example:
      CurrentEntity.Owner = Teams.Global

    An alternate action is not required.

     

  4. Click OK twice to exit the editor.
