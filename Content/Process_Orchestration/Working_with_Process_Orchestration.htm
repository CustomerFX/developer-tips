Implementing Process Orchestration

Implementing Process Orchestration involves the following basic tasks:

  1. Set up the Process Host portal with Application Architect. Refer to "Deploying the Saleslogix (Web Client) portal" in the Application Architect online Help and follow the steps appropriate for your situation to complete the following:

  2. Deploy the Process Host portal

  3. Configure IIS settings for Process Host

  4. Test the Process Host portal
