Upgrading Customized Help

When upgrading from v7.5.2 and later your customizations will be captured in the bundle process. However, when upgrading from v7.5.0 or v7.5.1 you must recreate your customizations in the new help topics.

In version 7.5.2, the Web Help was converted from .aspx to .htm. If you have customized Web Help topics from a previous version, you must recreate your customizations in the new help files or recreate custom topics as new files. You should name the file the same name that you used in the previous version, but with the file extension .htm instead of .aspx.

To recreate your customizations

  1. Before upgrading your Web Host, save any customized Web Help files to a location outside the VFS.
  2. Upgrade your Web Host.
  3. Browse to the location where you saved your customized Web Help files before upgrading.
  4. Do one of the following:
    • If the topic is a Web Help topic that is provided by Saleslogix, but contains customized content, follow the steps for adding customized content to the help file that was installed during the upgrade.
    • If the topic is a custom topic that you created and was not provided by Saleslogix, follow the steps for creating a custom help topic.
