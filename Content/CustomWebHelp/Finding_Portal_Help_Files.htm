Finding Portal Help Files

The Help files for each portal are stored separately in the location where the Web portal was deployed. By default:

  • Saleslogix (Web Client) portal Help files are stored in - \\Inetpub\wwwroot\SlxClient\help
  • Saleslogix Customer portal Help files are stored in - \\Inetpub\wwwroot\SlxCustomerPortal\help

The source of these deployed files is stored in the Portal Manager in the Application Architect project. Updates to help files should be copied here because deployed files are overwritten by each deployment.
