Linking a Help Topic to a Help Button

You can create a custom help topic for a custom form or Quick Form and link it to a help button so that it looks like any other Saleslogix Web view.

To link a help topic to a help button

  1. Create an alias for the help topic.
    If you give the alias the same name as the Smart Part, then the help button will be linked without having to set the Help Topic property in step 2.
  2. If the alias name is not the same as the Smart Part name, then you must set the Smart Part Help Topic property. See "Smart Parts Settings" in the Application Architect online Help.
    1. Open the page that contains the Smart Part where you want to add the Help button in the Page Editor. See "Using the Page Editor" in the Application Architect online Help..
    2. On the Smart Parts tab, click the Smart Part in the list
    3. In the Help Topic box type the name of the alias you created in step 1.

      If you are linking to a topic in a child file, use the alias name from the child file alias.xml.

    4. Click Save.
  3. When you have made all of your custom changes, deploy the changes to your Web site.
