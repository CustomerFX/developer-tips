Deleting Topics from a Project

You can delete help topics that do not apply to your business. In addition to removing the topics, you should remove any links in the table of contents, index or other topics.

To delete

  1. Identify which topics you want to delete.
  2. In the Application Architect, open the portal from which you want to delete the help topic.
  3. In the Project Explorer tree view, expand SupportFiles.
  4. Expand help, and then expand Content.
  5. Do one of the following:
    • To delete individual topics, right-click the topic to be deleted and select Delete File. Repeat for each topic to be deleted.
    • To delete a folder and any contents, right-click the folder and click Delete Folder. Repeat for each folder to be deleted.
  6. When you have finished deleting files or folders, click Save All.
  7. To remove links to deleted topics from the table of contents:
    1. In the Project Explorer tree view, expand SupportFiles\help\Data.

    2. Double-click TOC.xml.

    3. Search for and remove any links to topics you removed in step 6. To remove a link, delete everything between and including the < and >. For example, you would remove this entire line:

      4. <TocEntry Title="Deleting an Account" Link="/Content/Accounts/Test.htm" />

    4. Repeat steps a-c for each table of contents link you want to delete.
    5. When finished, save your changes.
  8. To remove links to deleted topics from the index:
    1. In the Project Explorer tree view, expand SupportFiles\help\Data.
    2. Double-click index.xml.
    3. Search for and remove any index entries to topics you removed in step 6. To remove an entry, delete everything between and including the <IndexEntry Term and</IndexEntry>. For example, you would remove this entire entry:

      <IndexEntry Term="Test">

      <IndexEntry Term="Test">

                  <Entries />

                  <Links>

                    <IndexLink Title="Test" Link="/Content/Test/test.htm#kanchor10000" UsedConditions="" />

                  </Links>

                </IndexEntry>

    4. Repeat steps a-c for each index entry that you want to delete.
    5. Count the number of index entries you deleted.
    6. Decrease the number after <CatapultTargetIndex Count= (located near the top of the index.xml file) by the number of index entries you just deleted.
    7. Save your changes.
  9. Perform a search for and note any links to the topics you deleted.
    1. Browse to the generated help files for each portal located in \\Inetpub\wwwroot. For example \\Inetpub\wwwroot\SlxClient\help
    2. Right-click the help folder and select Search.
    3. In the Search Results left pane, in the A word or phrase in the file box, type the name of the deleted file including the".htm" file extension. For example, slxoverview.htm
    4. Click Search.
    5. Open each file and remove any links to the deleted help topic.
    6. When you have finished deleting links, click Save All.

    7. Repeat steps a-g for each help topic you deleted.

  10. To remove any help buttons that link to a deleted help topic:

    This step is not necessary if the form with the help button has also been deleted.

    • For a custom form follow the steps for Linking a Help Topic to a Custom Form.
    • For a quick form follow the steps for Linking a Help Topic to a Custom Form.
  11. To delete a help topic from multiple portals, repeat steps 2 through 10 for each portal.

If you choose to remove Saleslogix help topics, you will need to repeat these steps each time you upgrade to a new version.
