Adding or Removing Images in a Help Topic

You can add images to a help topic to display a button or icon used in the Saleslogix Web interface.

To add

  1. In the Application Architect, open the portal for editing.
  2. In the Project Explorer tree view, expand Support Files\help\Content.
  3. Open the subfolder that contains the help topic you want to customize, and then double click the topic you want to edit.
  4. In the help file, add the following where you want to insert the image:
    5. <img src="../image/blubull.gif" />
  5. Change the blubull.gif to the file name of the image.
  6. If this is a new image file, you must add it to the portal.
    1. In the Application Architect, open the portal where you are adding the image file.
    2. In the Project Explorer tree view, expand Support Files, and then browse to the help\Content\image folder.
    3. Right-click images, select Add Existing Files, browse to the file you want to add, and then click Add.
  7. If you want the image to be hidden unless a keyword is clicked:
    1. Add <span class="MCExpanding"><a href="javascript:void(0);" class="MCExpandingHead" onclick="FMCExpand( this ); return false;"> in front of the text  to be clicked.
    2. Add </a> after the text to be clicked.
    3. Add <span class="MCExpandingBody" style="display: none;"> before the image code you added in step 1.
    4. Add the closing tags </span></span> after the image code from step 1.
  8. When you have finished adding images, click Save All.

    • Your changes will not be visible until you deploy the changes to your Web site.

    To remove an image

    1. In the Application Architect, open the portal for editing.
    2. In the Project Explorer tree view, expand Support Files\help\Content.
    3. Open the subfolder that contains the help topic you want to customize, and then double click the topic you want to edit.
    4. In the help file, select  the image you want to delete. For example, "<img src="../image/blubull.gif" />"
    5. Press Delete.
    6. When you have finished deleting images, click Save All.

    Your changes will not be visible until you deploy the changes to your Web site.
