Including a Topic in the Table of Contents

After creating a custom topic, you can customize the table of contents so that the topic appears in the table of contents.

Files in a help directory are specific to the portal where they are stored. Changes must be made in each portal separately.

To include a topic in the table of contents

  1. In the Application Architect, open the portal for editing.
  2. In the Project Explorer tree view, expand Support Files\help\Content.
  3. Open the subfolder that contains the help topic you want to edit, and then double click the help file.
  4. Include the following code, above the <head> tag:   
    <html xmlns:MadCap="http://www.madcapsoftware.com/Schemas/MadCap.xsd" MadCap:tocPath="Accounts">
  5. Replace Accounts with the name of the book you want the topic to appear under.
  6. Save your changes and close the help topic.
  7. In the Application Architect, open the portal for editing.
  8. In the Project Explorer tree view, expand SupportFiles\help\Data.
  9. Double-click TOC.xml.
  10. Find the table of contents entry that you want to precede the topic you are adding.
  11. Paste the following code directly below the previous table of contents item.
    <TocEntry Title="Deleting an Account" Link="/Content/Accounts/Test.htm" />
  12. Replace the Title with the title you want to appear in the table of contents and change the Link to point to the topic you added.
  13. When finished, save your changes.

Your changes will not be visible until you deploy the changes to your Web site.
