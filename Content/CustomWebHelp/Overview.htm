The Help files for each portal are stored separately in the location where the Web portal was deployed. By default:

  • Saleslogix (Web Client) portal Help files are stored in - \\Inetpub\wwwroot\SlxClient\help
  • Saleslogix Customer portal Help files are stored in - \\Inetpub\wwwroot\SlxCustomerPortal\help

Files in a help directory are specific to the portal where they are stored. For topics to be accessed by multiple portals, they must be copied to each portal. For example, for the SpeedSearch help topics to be available for the Saleslogix (Web Client) portal and Saleslogix Customer portal, they must be copied to \\Inetpub\wwwroot\SlxClient\help and \\Inetpub\wwwroot\SlxCustomerPortal\help respectively.

This book explains how to:

  • Upgrade customized help
  • Customize an existing Saleslogix Web help topic
