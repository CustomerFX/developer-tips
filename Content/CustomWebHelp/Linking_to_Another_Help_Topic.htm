Linking to Another Help Topic

It is often helpful to link from one topic to another related help topic. Use the steps below to insert a link to another help topic.

To link

  1. the Application Architect, open the portal for editing.
  2. In the Project Explorer tree view, expand SupportFiles\help\Content
  3. Open the subfolder that contains the help topic you want to customize, and then double click the topic you want to edit.
  4. Type or find the text you want to use to link to another topic.
    This is the text the user will click to open the linked topic.
  5. In front of the text add the following code:
    <a href="topicname.htm">
  6. Change topicname to the name of the help file you want to open when the text is clicked.
  7. If you are linking to a topic that is in a different folder, you must also include the folder name before the topicname.
    For example, if the help topic, Help\Content\Accounts\AccountDelete.htm, includes a link to Help\Content\Contacts\ContactDelete.htm, the code would be:

    <a href=" ..\Contacts\ContactDelete.htm">
  8. After the linked text, type </a>.
  9. When you have finished adding links, click Save All.

Your changes will not be visible until you deploy the changes to your Web site.

To remove a link

  1. In the Application Architect, open the portal for editing.
  2. In the Project Explorer tree view, expand Support Files\help\Content.
  3. Open the subfolder that contains the help topic you want to customize, and then double click the topic you want to edit.
  4. In the help file, find the link you want to delete. For example, "<a href="topicname.htm">"
  5. Press Delete.
  6. When you have finished deleting links, click Save All.

Your changes will not be visible until you deploy the changes to your Web site.
