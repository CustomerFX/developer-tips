Adding Custom Help Topics to a Project

The help is organized into topics and folders. Folders can be used to group related topics.

To add

  1. Create a new help file.
  2. In the Application Architect, open the portal to which you want to add the help topic.
  3. In the Project Explorer tree view, expand SupportFiles.
  4. Expand help, and then expand Content.
  5. If you need to add a new folder, right-click the help\Content folder, and then click New Folder, and enter a name for the folder, and click Save.
  6. Right-click the folder where the help topic will be added, select Add Existing Files, browse to the file you want to add, and then click Add.
  7. When you have finished adding files or folders, click Save All.
  8. After deploying the Web site, you can link the help topic to a help button.
  9. If you want a topic to be available in multiple portals, you will need to repeat steps 2 through 8 for each portal.

Your changes will not be visible until you deploy the changes to your Web site.
