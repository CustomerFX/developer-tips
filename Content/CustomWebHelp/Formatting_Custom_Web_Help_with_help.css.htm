Formatting Custom Web Help

Formatting within Web help topics is controlled by classes and rules defined within a cascading style sheet file (.css). Class styles can be applied to HTML tags and allow you set style attributes for blocks of text. For more information about cascading style sheets and class styles and rules, refer to the help system in your editor of choice. The .css file used for the Saleslogix help topics is named SALESLOGIX.CSS and is located in the help\Content folder found under each Saleslogix Web portal.

To format

  1. Use the formatting classes in the SALESLOGIX.CSS file. If you need to create a format that does not already exist, create the format as a new class.
  2. When editing or creating a help topic in an editor, make sure that the SALESLOGIX.CSS is available and located in the appropriate folder so that you can preview the help using the style sheet.
