Column Sparkline

Column sparklines have vertical bars for each data point.

Required Parameters  

 

 Parameter

Description

Default Value

d

Comma-separated list of values to plot. Integers recommended. Real values may cause character limit for all parameters to be exceeded.

none

type

Type of graph to plot = column.

none

Optional Parameters

 

Parameter

Description

Default Value

height

Height of the resulting image in pixels.

14

below-color

Color of bars for data points less than value of goal.

blue

above-color

Color of bars for data points great than or equal to value of goal.

red

transparent

Indicates if the background is transparent.

true

scale

Indicates if values of data points should be scaled. If false, then provided values must fall within 0-100.  If true, values will be scaled to 0-100.

false

bar-width

Width of each bar in pixels. Add this to bar-gap and then multiply by the number of bars to get the image width.

1

bar-gap

Space between bars in pixels.

1

goal

Goal amount.

0

goal-color

Color of goal line.

white

Examples

 

Sparkline

Query String

<img src="Libraries/SparkHandler.ashx?type=column&height=15&bar-gap=2&bar-width=4&goal=9&goal-color=green&scale=true&d=13,5,15,4,25,7">
