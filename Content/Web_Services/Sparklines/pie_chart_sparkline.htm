Pie Chart Sparkline

Pie Chart sparklines show each data point as a percentage of the total of all data points. Wedges represent data points in the clockwise direction, starting at 3 o'clock.

Required Parameters  

 

 Parameter

Description

Default Value

d

Comma-separated list of values to plot. Integers recommended. Real values may cause character limit for all parameters to be exceeded.

none

type

Type of graph to plot = pie.

none

Optional Parameters

 

Parameter

Description

Default Value

height

Height of the resulting image in pixels.

14

pie-colors

Color of pie segments, rotates through the list of colors.

red, orange, yellow, green, blue, purple, violet (rotates through list)

transparent

Indicates if the background is transparent.

true

Examples
