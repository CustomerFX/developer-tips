Bullet Chart Sparkline

   

A Bullet Chart sparkline uses a horizontal bar to represent a single data point. It can show color bands to indicate qualitative ranges and a target value. If the value of the width parameter is less than the height parameter, the bar is vertical.

Required Parameters  

 

 Parameter

Description

Default Value

type

Type of graph to plot = bullet.

none

value

Value to plot.

-1

scale-max

Upper boundary of graph range. Lower boundary is zero.

-1

Optional Parameters

 

Parameter

Description

Default Value

target

Target value

-1

height

Height of the resulting image in pixels.

14

width

The width of the resulting image in pixels.

200

first-band

Boundary between first and second background bands.

1/3 of scale-max

second-band

Boundary between second and third bands.

2/3 of scale-max

first-color

Color of first band.

dark gray

second-color

Color of second band.

light gray

third-color

Color of third band.

white

target-color

Color of target marker.

black

value-color

Color of value bar.

black
