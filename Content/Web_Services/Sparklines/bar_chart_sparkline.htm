Bar Chart Sparkline

    

Bar chart sparklines have horizontal bars for each data point.

Required Parameters  

 

 Parameter

Description

Default Value

d

Comma-separated list of values to plot. Integers recommended. Real values may cause character limit for all parameters to be exceeded.

none

type

Type of graph to plot = bars.

none

Optional Parameters

 

Parameter

Description

Default Value

bar-height

Height of each bar in pixels. Multiply by the number of bars to get the image height.

3

width

The width of the resulting image in pixels.

20

bar-colors

One or more bar colors. Separate colors with commas.

blue

shadow-color

Color of bar shadow.

#222222

align-right

Align right instead of left.

false

transparent

Indicates if the background is transparent.

true

scale

Indicates if values of data points should be scaled. If false, then provided values must fall within 0-100.  If true, values will be scaled to 0-100.

false

Examples
