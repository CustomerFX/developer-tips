Smooth Sparkline

Smooth sparklines have straight lines that connect data points.

Required Parameters  

 

 Parameter

Description

Default Value

d

Comma-separated list of values to plot. Integers recommended. Real values may cause character limit for all parameters to be exceeded.

none

type

Type of graph to plot = smooth.

none

Optional Parameters

 

 Parameter

Description

Default Value

step

Number of pixels in the width per data point.

2

height

Height of the resulting image in pixels.

20

min-color

If min-m = True, the color of the minimum point.

#0000FF

max-color

If max-m = True, the color of the maximum point.

#00FF00

last-color

If last-m = True, the color of the last point.

#FF0000

range-color

Color of a band highlighting the range of values. See range-upper and range-lower.

#CCCCCC

border-color

Color of the left and bottom borders of the image.

#0000FF

border-visible

Indicates if the border is shown as a line.

true

first-color

Color of the first part of a smooth line.

#102541

second-color

If changecolorat <> -1, the color of the second part of a smooth line.

#F28226

min-m

Indicates if the smallest Y-value is shown by a point (uses min-color).

false

max-m

Indicates if the largest Y-value is shown by a point (uses max-color).

false

last-m

Indicates if the last data point is shown by a point (uses last-color)

false

range-lower

If > 0, use Y=range-lower as lower edge of shaded band. Typically set to lowest Y-value of all data points so they are plotted within shaded range. Sometimes set to range-upper to make a "shaded" line instead of a band.

0

range-upper

If > range-lower, use Y=range-upper as upper edge of shaded band. Typically set to highest Y-value of all data points so they are all plotted within shaded range. Sometimes set to range-lower to make a "shaded" line instead of a band.

0

transparent

Indicates if the background is transparent.

true

scale

Indicates if values of data points should be scaled. If false, then provided values must fall within 0-100.  If true, values will be scaled to 0-100 unless scale-min and scale-max are set.

false

change-color-at

Number of the data point where the color of the smooth line changes to second-color. Use with type=smooth only.

-1

scale-max

If scale = True, this value is scaled to 100.

0

scale-min

If scale = True, this value is scaled to 0.

0

min-width

Minimum width of the image in pixels. Useful if you want to extend the image to fit a column.

50

label-ticks

Indicates if label markers min, max, last are displayed.

false

line-width

Width of the data line in pixels.

1

border-width

Width of the border in pixels.

1
