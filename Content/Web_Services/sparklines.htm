Sparklines

A sparkline is a small, high resolution graphic embedded in text. It can be used as a replacement for a number or a range of numbers.  

The Saleslogix Sparkline web service, SparkHandler.ashx, generates an image based on the parameters sent in a URL. The web service is also used by the quick form Sparkline Control. See "Sparkline Control" in the Application Architect online Help.. Saleslogix supports the following sparkline types:

  • Bar Chart

  • Impulse

  • Bullet Chart

  • Pie Chart

  • Column

  • Smooth

  • Discrete

 

 

All types except Pie Chart allow scaling of values. If values are not scaled, then they must be between 0 and 100 except Impulse. Impulse values must be between -50 and 50. If scaling is on, then the highest value is scaled to 100. With the Smooth type, you can set a minimum and maximum to scale to.

To add a Sparkline control to your custom form

  1. Edit a custom smart part. See "Editing a Quick Form" in the Application Architect online Help.

  2. Add an image tag to the code-behind according to the following format:

<img src="Libraries/SparkHandler.ashx?querystring">
 

Refer to the individual sparkline type for the parameters of the querystring:

  • Bar Chart

  • Bullet Chart

  • Column

  • Discrete

  • Impulse

  • Pie Chart

  • Smooth

  1. Click Save.

Example 1 - create a Smart Part with a static URL

  • Create a new custom Smart Part and paste the following lines into the code:

<%@ Control Language="C#" AutoEventWireup="true" CodeFile="test.ascx.cs" Inherits="test" %>

<img src=Libraries/SparkHandler.ashx?type=smooth&d=1,2,3,4 />

Example 2  - Review the CampaignResponses Custom Smart Part

  1. Under Portal Manager in the Project Explorer, expand Saleslogix (Web Client) portal > Pages.

  2. Right-click Welcome Page , and then click Edit Page.

  3. Click the Smart Parts tab.

  4. Select CampaignResponses (Custom), and then click Edit.

  5. Search for code containing "imgURL" and then "sparkdata".

  6. To see a rendering of this Smart Part, do the following:

  1. Open the Welcome page of the Saleslogix Web Client, and click the down-arrow next to My Workspace.

  2. Click Add Content.

  3. Select Campaign Responses, and click OK.

  4. Right-click the image, and click Properties to see the URL of the sparkline.
