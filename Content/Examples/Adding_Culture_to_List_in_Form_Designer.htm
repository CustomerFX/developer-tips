Adding a Culture to List in the Form Designer

The Form Designer in the Saleslogix Web Client has a Culture drop-down list which displays a static list of cultures. You can add to the list by editing the base.master file.

To add

  1. Open the SLXClient portal for editing. See "Editing a Portal" in the Application Architect online Help.
  2. Expand SupportFiles > Masters.
  3. Right-click Base.Master, and then click Open.
  4. Add the following code to the bottom of the file, directly after the last script tag. Substitute your culture code(s).

    <script type="text/javascript">

    require(['Sage/QuickForms/Design/DesignPanel'], function(DesignPanel) {

         DesignPanel.prototype.cultureList.push('fr-FR', 'es-ES');

    });

    </script>

  5. Click Save and close the file.
  6. Redeploy the portal.
