Setting the Maximum File Upload Size for Web Applications

You can alter the maximum size of files uploaded to the Saleslogix Web Client. The default size is 4MB.

To modify

  • Edit the web.config file with the following changes:
    1. In the httpRuntime section, add the maxRequestLength attribute.
    2. Type the default size using the following format.
      <httpRuntime requestValidationMode="2.0" maxRequestLength="5000" />
