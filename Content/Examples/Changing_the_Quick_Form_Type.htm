Changing the Quick Form Type

You can change the type of a quick form by editing the XML file.

To change

  1. In Application Architect, verify that the quick form you want to edit is not open in the Quick Form Designer.
    The form must be closed to prevent overwriting your changes to the raw XML file with changes on the form.
  2. In the model, open the quick form XML file. If the project workspace is stored in the Virtual File System, use VFS Explorer. Otherwise, browse to it on the local disk under the project workspace folder.
    The form is located under: Model > Entity Model > package > entity > QuickForms.
  3. Find the <Property name> node near the top of the file.
  4. Change the view type to Sage.Platform.QuickForms.QuickFormSummaryViewDefinition to make the quick form a summary view or to Sage.Platform.QuickForms.QuickFormMainDetailViewDefinition to make it a main detail view. For example:
    <Property name="DefinitionType" type="System.String"> Sage.Platform.QuickForms.QuickFormSummaryViewDefinition, Sage.Platform.QuickForms</Property> sets the quick form to be a summary view.
  5. Save the file.
  6. In the Project Explorer, right-click the project name containing the quick form, and then click Reload Project.
    This ensures that the form definition is updated in the project.
