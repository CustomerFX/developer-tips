Loading Multiple Versions of jQuery

You can load multiple versions of jQuery by using the noConflict function.  For more information on the function, see http://api.jquery.com/jQuery.noConflict/.

Saleslogix assumes the global $ is the version of jQuery that shipped with the product. To load a new version of jQuery, use a different variable/namespace.

Example:

    <%--Load new version of jQuery 1.7.2 --%>

    <script pin="pin" language="JavaScript" src="Libraries/jQuery/jquery-1.7.2.js" type="text/javascript"></script>

    <script pin="pin" language="JavaScript" src="Libraries/jQuery/jquery-ui-1.8.18.custom.min.js" type="text/javascript"></script>

    

    <%--Unload new version, preserve it in a jQuery172 global --%>

    <script pin="pin" type="text/javascript">

        window.jQuery172 = $.noConflict(true);

    </script>

    

    <%-- Load jQuery that came with SalesLogix to preserve $ --%>

    <script pin="pin" language="JavaScript" src="Libraries/jQuery/jquery.js" type="text/javascript"></script>

    <script pin="pin" language="JavaScript" src="Libraries/jQuery/jquery-ui.js" type="text/javascript"></script>
