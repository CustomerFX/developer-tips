Changing the Logo in the Saleslogix (Web Client) portal

You can replace the Saleslogix logo image used in the Web Client or point it to a different image. The default logo is 134 x 23 pixels. If you are using a different image size, review the client screens for any impact to their layouts.

To replace

  1. Name your .gif file saleslogix.gif.
  2. Open the SLXClient portal for editing. See "Editing a Portal" in the Application Architect online Help.
  3. Replace the saleslogix.gif file under SupportFiles\images with the new one.
  4. Redeploy the portal.

To point to another image

  1. Open the SLXClient portal for editing. See "Editing a Portal" in the Application Architect online Help.
  2. Expand SupportFiles > Images, and then add the new image. See "Adding Existing Files" in the Application Architect online Help.
  3. Expand SupportFiles > Masters, and then do the following steps for the Base.Master and dashboard.master files:
    1. Right-click the file, and then click Open.
    2. Find the following line and change the relative url to point to another image of type .gif, .jpg, or .png:
      <asp:Image runat=”server” ID=”imgLogo” style=”float:left” ImageALign=”Middle” ImageUrl=”~/images/saleslogix.gif” AlternateText=””/>
    3. Click Save and close the file.
  4. Redeploy the portal.
