Working With Static versus Dynamic Entities through Code

Each static entity has an interface with well-defined properties. Since dynamic entities do not, they must be manipulated with strings and dictionaries. If you reflect on an entity class to see its list of properties, you cannot see dynamic properties in the list. The following two examples illustrate the difference between creating a new account statically and dynamically:

STATIC:

 

    var account = EntityFactory.Create<IAccount>();

    account.AccountName = "Abbott";

    account.Owner = EntityFactory.GetById<IOwner>("SYST00000001");

    account.Save();

DYNAMIC:

 

    var account = (IDynamicEntity) EntityFactory.Create("Account");

    account["AccountName"] = "Abbott";

    account["Owner"] = EntityFactory.GetById<IOwner>("SYST00000001");

    EntityFactory.Save(account);
