Redirecting Web Client Logoff/Timeout

By default, the Saleslogix Web Client redirects to the logoff.aspx page when it times out or logs off. You can edit the appSettings.config file to redirect to another page.

To redirect

  • Edit the appSettings.config file with the following changes:
    1. In the appsettings section, locate the value for the LogoffRedirectUrl key.
    2. Modify it. For example:
      <add key="LogoffRedirectUrl" value="~/logoff.aspx"/>
      where value can be a relative path such as ~/logon.aspx or any full URL such as http://www.mycompany.com
