Configuring the Saleslogix Cache Server

The Saleslogix Cache Server is used to store Web data which allows the cache to be real-time. By default, the Saleslogix Cache Server is installed on the Web Host. If you install the Saleslogix Cache Server on a server other than the Web Host, you must update the web.config file to include the location of the Saleslogix Cache Server.

To configure

  • Edit the web.config file with the following changes:
    1. In the Sage.Platform.Caches.Memcached section, locate the value for the host IP address.
    2. Modify it with the IP address of the server on which the SalesLogix Cache Server is installed. For example:

      <add address=“234.0.0.1” port=“11211” /
      The default “127.0.0.1” port=“11211” is set to the localhost of the Web Host Server.
