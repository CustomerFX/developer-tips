Supported Functions for SData Queries

Saleslogix supports the following functions in the SData specification for use in SData queries.  Refer tohttp://interop.sage.com/daisy/sdata/AnatomyOfAnSDataURL/QueryLanguage for more information.
 

String Function

Example Query

concat

/sdata/slx/dynamic/-/accounts?where=concat(Status,Type) eq 'ActiveCustomer'

left

/sdata/slx/dynamic/-/accounts?where=left(AccountName,2) eq 'Co'

length

/sdata/slx/dynamic/-/accounts?where=length(AccountName) eq 11

lower

/sdata/slx/dynamic/-/accounts?where=lower(Type) eq 'customer'

right

/sdata/slx/dynamic/-/accounts?where=right(AccountName,2) eq 'es'

substring

/sdata/slx/dynamic/-/accounts?where=substring(AccountName,3,2) eq 'te '

upper

/sdata/slx/dynamic/-/accounts?where=upper(Type) eq 'CUSTOMER '

Numeric Function

Example Query

abs

/sdata/slx/dynamic/-/accounts?where=abs(Revenue) lt 1000

floor

/sdata/slx/dynamic/-/accounts?where=floor(Revenue) lt 1000

round

/sdata/slx/dynamic/-/accounts?where=round(Revenue,-3) lt 1000

sign

/sdata/slx/dynamic/-/accounts?where=sign(Revenue) eq -1

trim

/sdata/slx/dynamic/-/accounts?where=trim(Type) eq 'Customer'

Date Function

Example Query

currentTimestamp

/sdata/slx/dynamic/-/accounts?where=currentTimestamp() gt ModifyDate

dateAdd

/sdata/slx/dynamic/-/accounts?where=dateAdd(ModifyDate, 2) gt CreateDate

dateSub

/sdata/slx/dynamic/-/accounts?where=dateSub(ModifyDate, 2) lt CreateDate

day

/sdata/slx/dynamic/-/accounts?where=day(ModifyDate) eq 3

hour

/sdata/slx/dynamic/-/accounts?where=hour(ModifyDate) eq 21

minute

 /sdata/slx/dynamic/-/accounts?where=minute(ModifyDate) eq 9

month

/sdata/slx/dynamic/-/accounts?where=month(ModifyDate) eq 7

second

/sdata/slx/dynamic/-/accounts?where=second(ModifyDate) eq 15

year

/sdata/slx/dynamic/-/accounts?where=year(ModifyDate) eq 2012
