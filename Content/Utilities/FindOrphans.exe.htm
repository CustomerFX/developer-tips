FindOrphans.exe

The Find Orphans utility identifies code snippet orphans that may exist in your code. You can use this utility to find code snippet orphans that you may want to delete. This can help resolve issues you may have building the code snippet library.

For information on the arguments of this utility, type the utility name at a command line prompt in the Saleslogix install directory on the Administrative Workstation.
