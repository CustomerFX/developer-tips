BundleDiff.exe

The BundleDiff.exe finds differences between two bundle manifests. For example, to preview customizations that might require merging during an upgrade, you can use this utility to compare a Saleslogix difference bundle with your customization bundle.

The utility writes output to the console which you can redirect to a log file. For information on the arguments of this utility, type the utility name at a command line prompt in the Saleslogix install directory on the Administrative Workstation.
