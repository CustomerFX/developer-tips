bundler.exe

The bundler.exe creates a bundle of part or all of a project at the command prompt using one of the following methods:

  • All - Bundles all items in the project.

  • Diff - Bundles all items that are different from another project.

  • Date - Bundles all items that have changed since the date provided.

  • Manifest - Uses a defined manifest file to create the bundle.

For information on the arguments of this utility, type the utility name at a command line prompt in the Saleslogix install directory on the Administrative Workstation. Refer to samplebundleconfig.xml for additional information on the configuration file you provide to the utility.
