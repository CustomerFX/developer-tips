Using Automation Tools

The following automation tools are available for use with your Saleslogix Web applications. These command-line utilities are installed in the Saleslogix directory on the Administrative Workstation.

  • BundleDiff.exe - Finds differences between two bundle manifests.

  • Bundler.exe - Creates a bundle of part or all of a project.

  • FindOrphans.exe - Finds code snippet orphans that may exist in your code.

  • FSCopy.exe - Imports and exports between Saleslogix-support file system types (VFS, ZIP, disk).

  • ModelUtility.exe - Automates command-line builds of the Sage.Entity.Interfaces.dll and the 'Web classes' of the associated implementation assemblies.
